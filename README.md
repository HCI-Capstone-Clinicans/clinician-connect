  ## Live Website
  https://hci-capstone-clinicans.github.io/clinician-connect/
  
  ## How To Run Code Locally
  1. Clone repository. Open terminal and run:
  ```
  git clone https://github.com/HCI-Capstone-Clinicans/clinician-connect.git
  ```
  2. Navigate into project folder
  ```
  cd ~/clinician-connect
  ```
  3. Install dependencies
  - Install Node.js if you haven't already: https://nodejs.org
  - Then on terminal run:
  ```
  npm install
  ```
  5. Start development server
  ```
  npm run dev
  ```
  6. Open in browser
  ```
  http://localhost:5174/clinician-connect/
  ```
